About Me

My name is MacKenzie Garrett and I have been coding for almost two years. I know how to code in Python, Python with turtle, HTML, CSS, and JavaScript. I am currently a receptionist at Great Clips and I was an intern for MAXXPotental. During my internship at MAXXPotental I had a variety of roles including: Network Incident Response Specialist, Policy Administrator, and Software Developer.

Receptionist: As a receptionist my job is to put people on the waitlist, do laundry, answer the phone, and make sure the salon is clean.

Network Incident Response Specialist: As a Network Incident Response Specialist I kept an eye on the network and resolved any problems that came up. I also helped my team with documentation and organization. The project I was assigned to was to create and implement a password management system.

Policy Administrator: As a Policy Administrator I made sure my team was within the law and helped create/ organize all of the data. The project I worked on while I was in this position was to find a way to collect data from the company website and make only the necessary information available to the proper teams.

Software Developer: As a Software Developer I created frame works for a company website. I also helped write the code for the website. The assignment I worked on while I was a Software Developer was to create a fully functioning website for a power company in a two week sprint.
